# 任务执行记录模板

> 每个 P0/P1/P2/P3 任务都要创建一份执行记录，按规划→沟通→执行→复盘四环节记录。

---

## 📋 文档定位说明

| 字段 | 内容 |
|------|------|
| **文档定位** | 单个任务的「全流程档案」 |
| **核心作用** | ① 强制把每个任务拆成规划、沟通、执行、复盘四步，避免跳过关键思考 ② 保留完整上下文，方便后续复盘和知识复用 ③ 证明你具备独立交付复杂任务的能力 |
| **使用场景** | ① 启动任何 P0/P1/P2/P3 任务时复制本模板 ② 任务执行过程中随时补充执行日志 ③ 任务结束后做复盘 ④ 向 mentor/面试官展示某个任务是如何完成的 |
| **维护责任** | Jasper 主责，AI 协同终端辅助；任务启动时创建，任务结束时归档 |
| **迭代规则** | ① 每个任务复制一份，存到 `01_execution/任务编号_任务名/` 下 ② 发现某个字段重复无用时删除，发现缺失时补充 ③ 同类任务执行3次后，沉淀为 Skill 或 SOP |
| **关联文件** | [决策日志模板](./决策日志模板.md) · [周报复盘模板](./周报复盘模板.md) · [能力整改看板](../能力整改看板.md) |
