# 01 · 初始化项目 Initialize Project

> 对应方法论：`codex init my-agent && cd my-agent` —— 创建项目目录，初始化基础结构，生成默认配置文件和目录。

PTA 是已存在的项目，这一步是补记：记录这个 Agent 最初为什么要做、解决什么问题。
新 Agent 立项时，这里应该是第一个写文件的地方。

## 本文件夹内容

- `需求定义.md` —— PTA 最初的需求定义文档（早于 S01-S05 代码本身，记录了立项动机）

## 后续 Agent 复用本模板时

1. 新建 `my-agent/` 项目根目录
2. 在这里写一份需求定义（要解决什么问题、目标用户是谁、成功标准是什么）
3. 再进入 `02_配置项目_Configure_Project/` 做全局配置
